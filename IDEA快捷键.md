#### 快捷键

~~~shell
Ctrl + Tab       		  #切换打开的文件窗口
Ctrl + W         		  #选中当前词
Ctrl + N	      		  #搜索类(Ctrl + Shift + T)
Ctrl + Alt + Shift + N     #快速找到Controller方法(/bill)
Alt + F7		 		  #搜索所有的引用处
Ctrl + Shift + F           #搜索所有文本出现的地方(Ctrl + H)
Ctrl + Shift + R           #替换所有文本出现的地方
Ctrl + Shilt + U           #大小写转换(X,Y)
Alt + Shilt + UP/DOWN      #上移下移
Ctrl + Alt + L			  #格式化(Ctrl + Shift + F)
Ctrl + B			      #找到类,方法,变量声明的地方(F3)
Ctrl + Alt + B			  #找到类,方法实现的地方
Ctrl + H				  #查看类的结构(F4)
Ctrl + F12				  #快速展示类的元素	
Ctrl + Alt + right/left	   #返回光标的位置
Ctrl + Shilt + backspace   #返回上次修改的位置
Ctrl + E				  #查看最近修改过的文件列表
Ctrl + G				  #快速到达某一行(Ctrl + L)
Ctrl + R			      #查找替换
Ctrl + Alt +  鼠标左键      #点击进去实现方法
Alt + Shift + F10          #run
Ctrl + Q 				  #查看方法的注释
Ctrl + F4 				  #关闭当前打开的类
Shift + Esc 			  #隐藏活动窗口
Ctrl + Shift + Enter       #结束当前语句
Alt + Enter                #get/set方法
Shift + Shift
Shift + F6                 #重命名你类名,方法名,变量
ctrl + U				 #跳到父类接口

#阅读代码的快捷键
ctrl+shift+.               #创建任意代码折叠块
ctrl 加上一个+ 		       #打开任意代码折叠块
ctrl+]  ctrl+[			  #大括号匹配
ctrl+shift+f7 结合f3       #可以高亮某个变量，而且随着鼠标的移动，这个高亮是不会消失的(这个很重要)。然后使用f3找到下一个使用该变量的地方
ctrl+shift+i		   	  #当你阅读的代码很长的时候，突然想看代码里某个类的定义,就可以在当前类里再弹出一个窗口出来



Ctrl + Alt + I 自动缩进
Ctrl + Alt + O 整理import
~~~
